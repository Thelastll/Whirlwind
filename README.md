### Начало работы
Для начала необходимо ввести ipv4-адрес, записывающийся через ".", принадлежащий к локальной сети.
ВАЖНО, передавать сообщения получится только в этой сети. Также чтобы принимать сообщения приложение должно быть постоянно запущеным.
Если адрес существует на компьютере, запустится основное окно.
Ip-адрес можно поменять нажав на соответствующую кнопку и ввести новый валидный адрес.

### Настройки
При нажатии на кнопку с шестеренкой откроется отдельное окно с настройками.
Они сохраняются автоматически, сразу после внесения изменений.

### Добавление собеседника
Для добавления нужно нажать на кнопку "+" и ввести уникальный ip-адрес и уникальное имя.
Для такого ввода не обязательно вводить существующий адрес, но чтобы сообщение дошло до адресата, адрес должен ему принадлежать.
Также при клике пкм по адресату его данные можно будет изменить или удалить.

### Отправка и получение сообщений
При клике на адресата он будет выбран как текущий собеседник и ему можно будет отправить сообщение.
Сообщение прописывается в строке ввода, а файлы прикрепляются через отдельную кнопку рядом.
Также при нажатии сочетания ctrl+v, данные из буфера обмена будут прикреплены.
При клике на кнопку отправить справа снизу или enter будет отправлено сообщение из строки ввода и все прикрепленные файлы.
Для добавления переноса строки в сообщение используется сочетание клавиш shift+enter.
При клике пкм на любое сообщение можно удалить его или копировать.
Для файлов также можно посмотреть их расположение. 
И если удаленный файл лежал не в папке проекта с сохраненными файлами, он не будет удален. 
А при копировании файла, будет скопирован глобальный путь до него.
Если кликнуть лкм по файлу, он откроется.
Любые изображения отображаются сразу при получении.
По ссылкам, начинающимся с http:// и https:// можно перейти.

### Взаимодействие с уведомлениями
У конкретного адресата можно выключить уведомления или звук или все вместе.
При клике на уведомление будет открыт диалог с написавшим.
Громкость уведомлений можно изменить в настройках.
